Een Eenvoudige To-Do Lijst App Maken met Python

Iedereen houdt van lijstjes! Een to-do lijst helpt je georganiseerd te blijven.
In deze tutorial bouwen we samen een basis to-do lijst app. Je leert hoe je een programma maakt waarin je taken kunt toevoegen, bekijken, markeren als voltooid en verwijderen.

Benodigdheden

• Een computer met Python geïnstalleerd. Je kunt Python downloaden via python.org.
• Een teksteditor (bijvoorbeeld Visual Studio Code, PyCharm of de ingebouwde IDLE van Python).

Stap 1: Het begrijpen van onze app

De app moet:
1. Gebruikers toestaan om nieuwe taken toe te voegen.
2. De lijst met taken weergeven.
3. Gebruikers toestaan om een taak als voltooid te markeren.
4. Gebruikers toestaan een taak te verwijderen.
5. Het programma blijft draaien totdat de gebruiker ervoor kiest om te stoppen.
