Tutorial: Hoe maak je een interactieve takenlijst in Python?

In deze tutorial zal worden uitgelegd hoe je in Python programmeert. Dit wordt gedaan aan de hand van een programma dat de basispunten van coderen aanleert. Met dit programma zal je uiteindelijk een takenlijst kunnen beheren.

Stap 1: Een string printen waarin de lijst is geschreven

Het eerste wat je programma moet doen, is een overzicht geven van wat de gebruiker kan doen. Hiervoor gebruiken we de print-functie. Met print kun je tekst of gegevens in de console tonen. In ons geval willen we de opties tonen die de gebruiker heeft.

            Python:
            
            print("opties:")
            print("- show")
            print("- add")
            print("- remove")
            print("- stop")

Wat gebeurt er hier?

print("tekst"): Dit toont tekst tussen de aanhalingstekens in de console. In dit geval laten we de opties zien die de gebruiker kan kiezen.

Stap 2: Een input schrijven

Nu de gebruiker de opties ziet, willen we dat ze een keuze kunnen maken. Hiervoor gebruiken we de input-functie. Met input kun je een vraag stellen en de invoer van de gebruiker opslaan. 

            Python:
            
            choice = input(":>")

Wat gebeurt er hier?

input("tekst"): Met input kan je de gebruiker iets laten typen in de console zodat het ergens anders kan worden gebruikt. Ook kun je van tevoren wat tekst toevoegen die het programma iets laat zien voordat je de invoer geeft, zoals wat voor invoer er wordt verwacht.

Stap 3: Met een if-statement checken wat de gebruiker heeft opgeschreven

            Python:
            
            if choice == "show":
                print("knippen", "graven", "schoonmaken")

Wat gebeurt er hier?

if choice == "show": Controleert of de invoer van de gebruiker gelijk is aan "show". Let op dat als Python iets moet controleren je een dubbel =-teken gebruikt.

Stap 4: De else en elif toevoegen om een reactie te geven als de input niet geldig is 

            Python:
            
            elif choice == "add":
                todo.append(input("wat wil je toevoegen?"))
            elif choice == "stop":
                programm_running = False
            else:
                print("no option detected")

Wat gebeurt er hier?

elif: Controleert een andere mogelijkheid, zoals "add". Hij zal alleen worden uitgevoerd als de voorgaande statements niet geldig zijn.

else: Voert de code uit als geen enkele voorgaande optie overeenkomt. Hier geven we een melding dat de invoer ongeldig is.

Stap 5: De while-code en de bijbehorende variabele maken

            Python:
            
            programm_running = True
            
            while programm_running:
                choice = input(":>")
                # De rest van de code komt hier

Wat gebeurt er hier?

programm_running = True: Een variabele die aangeeft of het programma moet blijven draaien.

Stap 6: De array aanmaken en hoe je deze kunt aanpassen

            Python:
            
            todo = ["knippen", "graven", "schoonmaken"]

Wat gebeurt er hier?

todo: De naam van de lijst.

Stap 7: De functie toevoegen zodat removechoice kan worden herhaald

            Python:
            
            def removechoice():
                removable = input("remove what?")
                if removable in todo:
                    todo.remove(removable)
                else:
                    print("input not found in list. try again:")
                    removechoice()

Wat gebeurt er hier?

def removechoice(): Dit definieert de functie.

Hoe werkt het volledige programma?

Het programma toont de opties en wacht op de invoer van de gebruiker. Afhankelijk van de invoer voert het programma acties uit: taken toevoegen, taken verwijderen, de lijst tonen of stoppen. De lus blijft herhalen totdat de gebruiker "stop" invoert.

Met dit programma kun je een interactieve takenlijst beheren! Je hebt nu de basisbeginselen van programmeren geleerd: variabelen, lijsten, functies, lussen en conditionele statements.
